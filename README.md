laravel-admin-messages
======================

[Demo]() use `username/password:admin/admin`

## Installation

```
$ composer require "laravel-admin-ext/messages @dev"

$ php artisan migrate

$ php artisan admin:import Tungnt\\Admin\\Message\\Messages
```

Open `http://your-host/admin/messages`
